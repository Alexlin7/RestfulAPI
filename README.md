### enable MailServer
* add **mail.properties** in resources/

![image](https://user-images.githubusercontent.com/81425837/178107245-9bbd996b-734d-47b5-a71f-bcc5843de911.png)

* and then write this
````
mail.host=
mail.port=
mail.enable_auth=
mail.enabled_starttls=
mail.protocol=
mail.username=
mail.password=
````
